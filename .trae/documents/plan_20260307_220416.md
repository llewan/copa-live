Para implementar la autenticación con Google (Gmail), realizaré los siguientes cambios en el frontend y backend.

### 1. Base de Datos (PostgreSQL)
Necesitamos adaptar la tabla `users` para permitir usuarios sin contraseña (autenticados externamente) e identificar usuarios de Google.
- **Acción**: Ejecutar una migración para:
  - Hacer que la columna `password` sea opcional (`NULLABLE`).
  - Añadir una columna `google_id` para vincular la cuenta de Google.
  - Añadir una columna `avatar_url` para guardar la foto de perfil de Google.

### 2. Backend (Node/Express)
- **Dependencias**: Instalar `google-auth-library` para verificar los tokens de seguridad de Google.
- **Nueva Ruta**: Crear endpoint `POST /api/auth/google` en `server/routes/auth.ts`.
  - Recibirá el token de identidad de Google desde el frontend.
  - Verificará la validez del token con Google.
  - Buscará si el usuario ya existe por su email.
  - **Si existe**: Inicia sesión y vincula la cuenta de Google si no lo estaba.
  - **Si no existe**: Crea un nuevo usuario automáticamente (sin contraseña) usando los datos de Google (email, nombre, foto).
  - Retornará el JWT de sesión de nuestra aplicación (igual que el login normal).

### 3. Frontend (React)
- **Dependencias**: Instalar `@react-oauth/google`.
- **Configuración**: Envolver la aplicación con `GoogleOAuthProvider` en `main.tsx`.
- **Componentes**:
  - Modificar `Login.tsx` para añadir el botón "Continuar con Google".
  - (Opcional) Modificar `Register.tsx` para añadir la misma opción.
- **Lógica**:
  - Actualizar `api.ts` para llamar al nuevo endpoint del backend.
  - Actualizar `authStore.ts` para manejar el inicio de sesión con Google.

### Requisitos del Usuario
Para que esto funcione, necesitarás obtener un **Google Client ID** desde la [Google Cloud Console](https://console.cloud.google.com/).
1. Crear un proyecto en Google Cloud.
2. Ir a "APIs & Services" > "Credentials".
3. Crear "OAuth Client ID" para "Web application".
4. Añadir `http://localhost:5173` (o tu puerto de frontend) en "Authorized JavaScript origins".
5. Copiar el **Client ID** y añadirlo a tus variables de entorno.
